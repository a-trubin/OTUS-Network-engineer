# EIGRP
## Цели:
1. Настроить EIGRP в С.-Петербург.
2. Использовать named EIGRP.
## Условия:
1. R32 получает только маршрут по умолчанию.
2. R16-17 анонсируют только суммарные префиксы.
3. Использовать EIGRP named-mode для настройки сети.
